Claude Code v2.1.120–123 Feature Bundle

Four Claude Code releases shipped April 28–29 with the highest feature density in weeks. v2.1.121 adds `alwaysLoad` to MCP server config (skip tool-search deferral entirely for trusted servers), `claude plugin prune` for orphaned deps, and PostToolUse hooks can now replace any tool's output via `hookSpecificOutput.updatedToolOutput` — no longer MCP-only. v2.1.120 ships `claude ultrareview [target]` for non-interactive CI pipelines, Windows PowerShell as git fallback, and `${CLAUDE_EFFORT}` env var usable in skill files. v2.1.122 adds `ANTHROPIC_BEDROCK_SERVICE_TIER` (default/flex/priority) and paste-a-PR-URL-into-/resume to find the originating session across GitHub, GitLab, and Bitbucket.
